# Symmetric Tree
Given the root of a binary tree, check whether it is a mirror of itself (i.e., symmetric around its center).
 
### Example 1:

![image](https://user-images.githubusercontent.com/61406986/222891816-4f8fe17f-018b-4607-932c-63efe388081b.png)

```
Input: root = [1,2,2,3,4,4,3]
Output: true
```
### Example 2:

![image](https://user-images.githubusercontent.com/61406986/222891852-ea4d37ae-15c6-4a8a-97d9-e07a5d6642c1.png)

```
Input: root = [1,2,2,null,3,null,3]
Output: false
``` 

### Constraints:

The number of nodes in the tree is in the range [1, 1000].<br>
-100 <= Node.val <= 100
